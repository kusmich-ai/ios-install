## BASELINE DIAGNOSTIC SYSTEM (4 Domains)

### **Administered During Onboarding (Day 0)**

Users complete brief validated assessments measuring 4 core domains. These establish the starting point for tracking transformation.

**Assessment Structure:**

1. **Calm Core Assessment** - Regulation Domain ~1 min
   - Measures perceived stress and autonomic load
   - Captures baseline regulatory capacity
   - Storage key: `baseline:calm_core`

2. **Observer Index** - Awareness Domain ~2 min
   - Measures meta-awareness and ability to observe thoughts
   - Directly assesses decentering capacity
   - Storage key: `baseline:observer_index`

3. **Vitality Index** - Outlook Domain ~1 min
   - Gold-standard positive-affect scale
   - Sensitive to change, quick administration
   - Storage key: `baseline:vitality_index`

4. **Focus Diagnostic** - Attention Domain (Part 1) ~1 min
   - Fast, reliable proxy for attentional control
   - Core metric for flow capacity
   - Storage key: `baseline:focus_diagnostic`

5. **Presence Test** - Attention Domain (Part 2) ~3 mins
   - Embodied attentional metric (counting breaths to 9, up to 5 cycles)
   - Test ends when user loses count or completes 180 seconds
   - Time-to-failure becomes the metric
   - Storage key: `baseline:presence_test`

**Total baseline time: ~8 minutes**

**Scoring:**
- Each assessment converts to 0-5 scale for consistency
- Attention domain = average of Focus Diagnostic + Presence Test
- Total baseline: 4 domains Ã— 5 = 20 points possible

**REwired Index Calculation:**
- Formula: (Regulation + Awareness + Outlook + Attention) / 4 Ã— 20
- Results in 0-100 scale
- Storage key: `baseline:rewired_index`

**REwired Index Tiers:**
- 0-20: System Offline (Critical)
- 21-40: Baseline Mode (Installing...)
- 41-60: Operational (Stabilizing)
- 61-80: Optimized (Coherent)
- 81-100: Integrated (Embodied)

**If User Agrees:**
Run the complete **Interpretation Audit (5-Step Debug)** from sub-protocol instructions:

**Step 0 (When Needed): Ground First**
- Assess nervous system state (sympathetic activation vs dorsal shutdown)

- Don't attempt cognitive work until system is in window of tolerance
